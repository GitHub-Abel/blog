最是那盖世无双金甲战士

2015年12月25日

作为一部国产动画，本片创作理念却显著带有着西化基因。光剧情上就是很典型的好莱坞模式：一位昔日大英雄，遭遇挫折陷入困境，在同伴路人潜移默化点拨之下，卧薪尝胆久经磨砺，终于找回自我王者归来的故事。人物设定上更是一改西游记的传统形象，这杂毛猴子乍一看分明就是标准的街头小混混古惑仔嘛！而大反派的造型无论人形状态还是怪兽形态又隐隐带着日漫的影子。
